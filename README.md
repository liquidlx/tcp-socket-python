# tcp-socket-python
A simple guide to create a TCP socket in Python

## O que é o modelo cliente-servidor?
Basicamente neste modelo temos dois lados: o cliente e o servidor (Thanks, Mr. Obvious).
São maquinas que normalmente são ligadas por uma rede, a diferença entre as duas é que o servidor é um *host* que está executando um ou mais serviços ou programas que compartilham recursos com os clientes. Um cliente não compartilha qualquer de seus recursos, mas solicita um conteúdo ou função do servidor.

## E o que é Socket??
Socket é um mecanismo de comunicação, usado normalmente para implementar o modelo cliente/servidor explicado acima, que permite a troca de mensagens entre os processos de uma máquina/aplicação servidor e de uma máquina/aplicação cliente. O bom de utilizarmos python para esse exemplo é que ele já vem com um modulo de socket padrão, assim não precisamos nada mais que o python instalado na nossa máquina.
